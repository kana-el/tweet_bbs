# tweet_bbs
個人練習用の自作PHP

# 概要
コミュニケーションの為のつぶやき掲示板を作成しました。
登録者用ページと管理者ページにそれぞれログインすることができます。

# 使い方
・メインページではつぶやきのみが見れます。

・登録者ページでは以下のことが可能です。
　・つぶやきを投稿。
　・自分のメッセージを削除。
  ・他者のメッセージに返信、いいねを押す。
  ・自分のアカウントの編集、退会。
 
 ・管理ページでは以下のことが可能です。
 　・メッセージ削除。
   ・登録アカウント編集、削除
